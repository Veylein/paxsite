Pax Privacy Policy

Pax stores only the minimum data required to make the bot function:

  • User ID (for leveling, pets, stats)
  • Guild ID (for server-wide settings)
  • Game data (items, chi, achievements, etc.)

Pax does NOT collect or store:

  • Messages
  • Passwords
  • Emails
  • Personal identifying information

You may request deletion of your Pax data at any time via the support server.
